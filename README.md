# Santa Monica Zen Center Website

This repository contains the restored original website for the Santa Monica Zen Center (smzen.org).

## Project Overview

This is a restoration of the original SMZC website, preserving the historical design and content as it appeared in the mid-2000s. The site has been prepared as a static deployment ready for modern hosting platforms.

## Site Structure

- **Landing page**: `index.html`
- **Main content**: Vision, training information, contact details
- **Calendar archive**: Historical monthly calendars from 2005 in `/calendar/`
- **Media**: PDF documents for training materials and forms in `/media/`
- **Images**: All original site graphics and photos in `/images/`

## Historical Note

This website represents the Santa Monica Zen Center as it operated in the early-to-mid 2000s. All content has been preserved as originally created, including calendar entries from 2005 and training materials from that era.

## Deployment

This is a static HTML site designed to be deployed on platforms like Vercel, Netlify, or similar static hosting services. All relative paths have been maintained to ensure proper functionality.
